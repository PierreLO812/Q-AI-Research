## đź”„ The 6-Phase Pipeline

When you run `start.exe`, the following sequence occurs:

### [1/6] Semantic Parsing
- **What happens**: The LLM/NLP bridge identifies physical constants (`gamma`, `hbar`) and operators.
- **Success Look**: `-> Extracted X physical entities.`

### [2/6] Physics Simulation
- **What happens**: The C++ core runs a Lindblad solver. 
- **Success Look**: `Lindblad simulation reference generated (11 data points).`

### [3/6] 'Le Cerveau' (Discovery)
- **What happens**: The engine generates thousands of candidate ASTs (Abstract Syntax Trees).
- **Log Watch**: 
  - `[AST TYPE ERROR]`: Rejection of invalid matrix/scalar logic.
  - `[PG-SR DIM] REJECT`: Rejection due to incoherent units (e.g., adding meters to seconds).

### [4/6] Python SINDy Offloading
- **What happens**: The C++ master spawns `worker_sindy.py`. 
- **Log Watch**: `[PythonBridge] PySINDy Worker Output: {"discovered_equation": ...}`.
- **Note**: This is pure mathematical regression over the tensors generated in Phase 2.

### [5/6] 'Le Juge' RLVR & Lean 4
- **What happens**: The candidate equation is translated to a `.lean` file and compiled.
- **Tactic Loop**: The engine tries `aesop`, `simp`, `quantum_simp`, and `linarith` sequentially.
- **Fail Mode**: If all tactics fail, the engine captures the **stderr** from the Lean binary and routes it to the mutator.

### [6/6] LLM Mutation (The Loop)
- **What happens**: If Phase 5 fails, the local LLM is invoked.
- **Input**: The failing AST + the exact Lean 4 error.
- **Output**: A JSON mutation (e.g., `replace sigma_x with sigma_z`).

### đź”´ Red Flags (Fatal Crashes)
- `[CRITICAL FATAL] Python Environment is missing`: You need Python in your PATH.
- `[FATAL CRASH] C++ Parser could not interpret token`: The discovery script returned a character the C++ engine doesn't recognize yet.

### đźźˇ Mathematical Rejections
- `[PG-SR DIM] REJECT: Cannot add/sub [s] and [INCOHERENT]`: Your current equation is trying to add time (seconds) to a dimensionless number.
- `[AST TYPE ERROR] Cannot apply Hermitian conjugate to a SCALAR`: Logic error where `dagger` was applied to a number instead of a matrix.

### đź”µ Lean 4 Errors (The Juge's Verdict)
- `error: failed to synthesize 'Inhabited' instance`: The theorem is mathematically incomplete or use opaque types incorrectly.
- `unexpected token '('`: Syntax error in the generated Lean code (rare).
